Release Notes for Acoustic EC

Acoustic echo cancellation software library for STM32 and ST digital MEMS microphones

Copyright 2017 STMicroelectronics

 

What is Acoustic EC library?

  • Acoustic EC library implements the well known Speex Acoustic echo cancellation algorithm.
  • Acoustic EC allows the user to experience advanced AEC algorithms whith a minimum effort. A fully configurable system is provided to perform  AEC.

AcousticEC features

  • Advance AEC library.
  • Configurable Speex preprocess options:
    • denoiser
    • AGC
  • PCM input
  • Designed for optimal performance in applications using ST’s digital MEMS microphones (such as MP34DT01-M).
  • Easy portability across different STM32 MCU families thanks to modular architecture and STM32Cube
  • Free user-friendly license terms
The Acoustic EC Acoustic Echo Cancellation software library is an optimized software implementation for Acoustic Echo Cancellation based on the open source patent-free SPEEX software.

This Library includes one header file acoustic_ec.h and binary/object codes running on STM32F4 devices based on STM32 devices based on CortexM4F and CortexM7F cores. The naming format is acoustic_EC_CMx_C.a/.lib, where:
  • x: CortexMx core class (CM0, CM3, CM4 and CM4F)
  • C: compiler (IAR, Keil and GCC)

Update History

V2.0.1 / 12-July-2017

Main Changes

  • Added IAR 8.11 support.

Development Toolchains and Compilers

  • IAR Embedded Workbench for ARM (EWARM) toolchain V8.11 (backward compatible with IAR 7.x)
  • RealView Microcontroller Development Kit (MDK-ARM) toolchain V5.23 
  • Ac6 System Workbench for STM32 V2.1.0 

Known limitations

  • The provided library supports MCUs based on M4 and M7 cores with FPU unit.
  • A compiler warning is generated when using the library in IAR v 7.x. It doesn't affect library performances.

V2.0.0 / 8-March-2017

Main Changes

Development Toolchains and Compilers

  • RealView Microcontroller Development Kit (MDK-ARM) toolchain V5.23 + ST-LINK
  • Ac6 System Workbench for STM32 V1.13.1 + ST-LINK/V2

Known limitations

  • The provided library supports MCUs based on M4 and M7 cores with FPU unit.
